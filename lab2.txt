Тема: 
	Системы контроля версий.

Время на выполнение: 
	2 дня.

Задание:
	1. Работая в команде выбрать общедоступное хранилище репозиториев Git, которое будет использовано 
для выполнения последующих заданий.
	2. Любым членом команды создать репозиторий с именем practice_bukep.
	3. Каждым членом команды получить/синхронизировать репозиторий на локальную машину.
	4. Каждым членом команды создать каталог с наименованием по шаблон strudent_<номер студенческого билета>, где
<номер студенческого билета> - номер вашего студенческого билета.
	5. Каждым членом команды создать в корне репозитория файл с именем README.md в который записать имя созданного вами в п.4 каталога.
	6. В каталог созданный в п.4 добавить пустой файл с именем .gitignore.
	7. Выполнить синхронизацию изменений локального репозитория в удаленный. При возникновении конфликтов корректно их уладить.
	8. Подготовить файл отчета выполнения задания, в котором указать последовательность команд git, которые были использованы
для выполнения данного задания. Дать ответ на вопрос - для чего нужен файл .gitignore.
	9. Отчету дать наименование lab2.txt и разместить в каталоге студента.
	10. Выполнить синхронизацию изменений локального репозитория в удаленный.

Критерии выполнения:
	Пройти по ссылки репозитория и убедиться в наличии ожидаемой структуры и файлов. Открыть отчет студента и убедиться в корректности выполнения задания.

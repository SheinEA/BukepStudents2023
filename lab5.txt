Тема: 
	Консольное приложение, калькулятор. Часть 2 - рефакторинг.

Время на выполнение: 
	2 дня.

Задание:
	1. Перейти в локальный репозиторий и создать ветку от main с наименованием по шаблону lab5_<номер студенческого билета>.
	2. Перейти в каталог с созданным ранее приложение BUKEP.Student.Calculator и открыть его в Visual Studio.
	3. Реализовать вычисление выражений с учетом приоритета операций. Например, (2+2)*2 = 8
	4. Выполнить рефакторинг приложения с использованием знаний полученных в предыдущей работе.
	5. Выполнить синхронизацию изменений локального репозитория в удаленный.
	6. Создать pull request в ветку main.
	7. По завершению предоставить ссылку на pull request. 

Критерии выполнения:
	Пройти по предоставленной ссылки и провести code review. Проверить работоспособность приложения на нескольких контрольных примерах.
Проверить поведение приложения при некорректном вводе выражений/операция.
